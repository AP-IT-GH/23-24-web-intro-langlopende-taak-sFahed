Virtualization of server and Monitoring

Virtual Server Monitoring

Bewaak de gezondheid en beschikbaarheid van je gevirtualiseerde serveromgeving. Wij ondersteunen de realtime-monitoring van alle belangrijke virtualisatieplatformen, zoals VMware, ESXi, Microsoft Hyper-V en bijbehorende virtuele machines. Monitoring van virtuele servers is al geïntegreerd, zodat uw virtuele machines kunnen worden gemonitord met bijna geen extra set-up. Onze agents zelf kunnen ook worden uitgevoerd op een virtuele serverinfrastructuur binnen de virtuele appliance.

  • Geïntegreerd logboekbeheer stroomlijnt de bewaking van het Windows gebeurtenissenlogboek en verbetert de mogelijkheden voor probleemoplossing.
  • Breid de bewakingsmogelijkheden eenvoudig uit met Microsoft Exchange, Active Directory, Microsoft SQL Server en Skype for Business, zodat u verzekerd bent van holistisch overzicht.
  • Bewaak een breed scala aan Windows-besturingssystemen, van de nieuwste versies tot oudere edities zoals Windows XP en Windows Server 2003, voor uiteenlopende omgevingen.
  • Implementeer de monitoring agent moeiteloos met behulp van een eenvoudig uitvoerbaar bestand zonder afhankelijkheden, waardoor het risico van injectie van kwaadaardige code uit netwerkgegevens wordt beperkt.
